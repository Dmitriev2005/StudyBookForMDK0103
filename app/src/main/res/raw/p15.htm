Компонент WebView

 

Android позволяет создать собственное окно для просмотра веб-страниц или даже создать свой клон браузера при помощи элемента WebView. Сам элемент использует движок WebKit и имеет множество свойств и методов. В последних версиях используется движок от Chromium, но большой разницы в этом нет для простых задач.

Виждет WebView также может работать с CSS и JavaScript. WebView также обеспечивает работу с историей посещений и навигацией назад и вперед по страницам.

В WebView мы можем загрузить какой-то сайт с помощью протокола HTTP. Метод loadUrl() используется для загрузки каких-либо URL в компонент WebView.

webView.loadUrl("http://nt-orsk.ru ");

Так как приложение будет использовать интернет, необходимо установить разрешение на доступ к интернету в файле-манифесте.

<uses-permission android:name="android.permission.INTERNET" />

Если вы хотите загружать в WebView страницы не из интернета, а со своего приложения, то разместите нужные файлы в папке assets, например, assets/mypage.html. Доступ к файлу вы можете получить через конструкцию file://android_asset.

webView.loadUrl("file:///android_asset/mypage.html");

 

Также можно загрузить файл из папки res/raw:

webView.loadUrl("file:///android_res/raw/cat.html");

 

Иногда возникают в приложении проблемы с кодировкойв виджете, для установки кодировки используется

WebSettings settings = mWebView.getSettings();
settings.setDefaultTextEncodingName("utf-8");

 

Прежде чем мы начнем изучать WebView дальше, есть два важных момента, которые нельзя упустить из вида:

                 Поддержка JavaScript: JavaScript по умолчанию выключен в Webview. Следовательно, веб-страницы, содержащие JavaScript не будут работать должным образом. Чтобы его включить, используйте следующих фрагмент кода:

getSettings().setJavaScriptEnabled(true);

У WebView есть множество методов, которые позволяют добиваться полной функциональности как у обычного браузера - обновить страницу, перейти на предыдущую страницу и т.д. Часть методов представлена ниже:

      reload()

      goForward()

      goBack()

Так как приложение будет использовать интернет, необходимо установить разрешение на доступ к интернету в файле-манифесте.

 
<uses-permission android:name="android.permission.INTERNET" />
 

Там же в манифесте модифицируем строчку для экрана, удалив заголовок из нашего приложения (выделено жирным):

 
<activity android:name=".HelloWebViewActivity"
 android:label="@string/app_name"
 android:theme="@style/Theme.AppCompat.NoActionBar">
